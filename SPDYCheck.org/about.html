SpdyCheck is a free, open sourced tool to test any website for the presence of the SPDY protocol.

SPDYCheck is hosted by Zoompf.com. We take great care to prevent excessive requests, including extensive use of caching and user level rate limits.

Still, if you are seeing elevated levels of traffic from this tool, please contact us at support@zoompf.com and we will work with you to block the offending activity. If possible, please also examine the X-SPDYCHECK-CLIENTIP header value as this indicates the IP address of the user initiating the test.
